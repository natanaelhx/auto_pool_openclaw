# Analise das skills usadas como base

A `auto-pools` foi desenhada como skill standalone. As skills abaixo servem como referencia conceitual, mas nao sao dependencias runtime.

## defimax

Base conceitual para ranking DeFi, yield, LPs, vaults, wallet monitor, alertas e guardrails.

## liquidity-pool-monitor

Base conceitual para APR, TVL, pools, impermanent loss e shortlist.

## multichain-wallet-portfolio / wallet-portfolio-tracker

Base conceitual para saldo disponivel, exposicao atual, concentracao por ativo, chain e protocolo.

## crypto-assets-analytics

Base conceitual para regime do ativo, bull/bear case e risco macro.

## funding-rate-monitor / squeeze-monitor

Base conceitual para risco de pressao em BTC/ETH/SOL, volatilidade e squeeze.

## crypto-divergence-rsi

Base conceitual para RSI, divergencia, estrutura tecnica e lateralizacao.

## trading-signal

Base conceitual para saida operacional: entrada, invalidacao, risco, cenario e tamanho sugerido.

## delta-trade-nado-kraken

Base conceitual para watcher, estado, logs, dry-run, limites e guardrails operacionais.
